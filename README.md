# Zooniverse React Starterify

A minimal React JS application starter kit, based on [React Starterify](https://github.com/Granze/react-starterify).
If youâ€™re looking for a Redux starter kit, check [this out](https://github.com/zooniverse/zoo-reduxify)
## Usage

__Install the dependencies:__

`npm install`

__Test:__

```npm run test```

__Development mode with livereload:__

```npm run start```

__When you are done, create a production-ready version of the JS bundle:__

```npm run build```
